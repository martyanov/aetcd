Installation
~~~~~~~~~~~~

.. code:: bash

    $ python3 -m pip install aetcd3

Basic usage
~~~~~~~~~~~

.. code-block:: python

    import aetcd3

    client = aetcd3.client()

    await client.put('foo', 'bar')
    await client.get('foo')
    await client.delete('foo')
